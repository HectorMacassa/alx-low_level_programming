# 0x15-file_io

This repository contains C programs related to file input/output operations.

## Files

### 0-read_textfile.c

This program reads and prints the content of a text file to the standard output.

### 1-create_file.c

This program creates a new text file.

### 2-append_text_to_file.c

This program appends text to the end of a file.

### 3-cp.c

This program copies the content of one file to another
